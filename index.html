Conway's Game of Life

A game that isn't played. Divised by John Conway in 1970, where you define its initial state, then following three simple rules it will work from there.

  1. Any live cell with two or three live neighbours survives.
  2. Any dead cell with three live neighbours becomes a live cell.
  3. All other live cells die in the next generation. Similarly, all other dead cells stay dead.

It is Turing Complete, which as a result one can recreate the game with the use of the game itself. OTCA metapixel
